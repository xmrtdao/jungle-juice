# &#127796; Jungle Juice — La Fortuna, Costa Rica

**Art-forward smoothie shop at the base of Volc&aacute;n Arenal.**

Hand-crafted smoothies made with tropical fruit picked at sunrise: mango, papaya, coffee, protein, vegan &mdash; every flavor kissed by the rainforest.

## &#127758; Features

- Responsive single-page site with smooth scroll navigation
- Full-bleed hero with Costa Rica jungle imagery
- Wildlife gallery (toucan, sloth, red-eyed frog, capuchin, jaguar, hummingbird)
- Menu grid with 9 signature smoothies, tags & pricing in CRC
- Horizontal gallery of Costa Rica landscapes & local fruit
- Location details with Google Maps integration
- Scroll-triggered reveal animations & parallax effects
- Mobile-first responsive design
